=== Rebrand redirects (301) ===

https://www.polk-county.net/         → https://www.polkfl.gov/            (county CMS)
https://www.polk-county.net/sitemap_index.xml → https://www.polkfl.gov/sitemap_index.xml
https://www.polkcountyclerk.net/     → http://www.polkclerkfl.gov/        (note: http, not https, on the target)

polkfl.gov robots.txt still references `Sitemap: https://www.polk-county.net/sitemap_index.xml` — the old hostname is the Sitemap directive target, but it 301s to the new hostname so crawlers end up at polkfl.gov anyway. Stale content in robots, not an operational issue.
