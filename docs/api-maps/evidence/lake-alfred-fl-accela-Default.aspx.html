Owner applying for permits must provide an Owner builder Affidavit found here


If the permit has been issued but a revision is needed, you must email a revision letter to permits@mylakealfred.com to inform us that new documents have been uploaded.


Contractors and licensed professionals will need to create an application for contractor registration / renewal before you will be able to apply for permits.


Florida Statute 162.06(1)(b) A person who reports a potential violation of a code or an ordinance must provide his or her name and address to the respective local government before an enforcement proceeding may occur. You will need to Create an account and login to file a Code Enforcement Complaint.
