                                              Responsibilities

                                              • Issue building permits

                                              Documents and Forms

                                              Polk City processes permits on Polk County’s Accela system, you can access it here. Building permits are done electronically.  Instructions for permit applications and uploading documents in Accela can be found here. All documents and plans for permits must be uploaded into Accela for permit processing.
