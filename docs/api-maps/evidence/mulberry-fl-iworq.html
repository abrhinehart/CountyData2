Existing Animal Licenses

Search For A License:

Find information on an existing license. Please have your license number handy.

Online services include:
1
Click on the icon to begin.
2
Choose a 'search by' option from the drop-down menu.
3
Enter a keyword or number and press search. Click on the row of the license you would like to see more information for.
